What it does

This plugin allows you to display your best sellers on the main page of your website (in the center column) — just like featured products. Starting with v2.0.0, the centerbox also displays on the category-listing pages, if enabled in the admin's configuration.

You can control various aspects (such as image size and number of columns) of the best sellers center box from the admin's Configuration->Best Sellers:

Field Name Description
Show Best Sellers - Main Page Show the Best Sellers on the main page? Set to 0 to disable or set the sort order for the display.
Show Best Sellers - Main Page w/ Categories Show the Best Sellers on the categories' listings? Set to 0 to disable or set the sort order for the display.
Maximum Display of Best Sellers The maximum number of best sellers to list in an enabled center-box.
Best Sellers - Columns per Row The number of columns to display in an enabled center-box.
Best Sellers - Image Width The width of the Best Sellers images.
Best Sellers - Image Height The height of the Best Sellers images.
Best Sellers - Show Out-of-Stock? Show out-of-stock products in the Best Sellers centerbox?

Installation

There are no core-file overwrites in this plugin, but there are two template-override files; you should always backup your cart's database and files prior to making any changes.

  1. Unzip the plugin's package file and
    1. Rename the YOUR_ADMIN folder to match your custom admin folder's name.
    2. Rename the YOUR_TEMPLATE folder to match your store's current template.
  2. If you currently have a template-override version of the following files, merge those changes with the changes supplied by this plugin:
    1. /includes/templates/YOUR_TEMPLATE/templates/tpl_index_default.php
    2. /includes/templates/YOUR_TEMPLATE/templates/tpl_index_categories.php
  3. Copy the files provided by the plugin to your Zen Cart:
    1. /includes/languages/english/extra_definitions/best_sellers_reloaded.php
    2. /includes/modules/best_sellers_reloaded.php
    3. /includes/templates/template_default/css/stylesheet_best_sellers_reloaded.css
    4. /includes/templates/template_default/templates/tpl_modules_best_sellers_reloaded.php
    5. /includes/templates/YOUR_TEMPLATE/templates/tpl_index_default.php
    6. /includes/templates/YOUR_TEMPLATE/templates/tpl_index_categories.php
    7. /YOUR_ADMIN/includes/functions/extra_functions/init_best_sellers_reloaded.php
    8. /YOUR_ADMIN/includes/languages/english/extra_definitions/best_sellers_reloaded_definitions.php
  4. Log into (or refresh the screen on) your Zen Cart admin; the plugin's initialization script creates Configuration->Best Sellers.

Uninstall

Replace the backup copies of the files you copied during the installation, delete the new files and run the plugin's file /uninstall_best_sellers_reloaded.sql.
